[Exybox-CLI](https://www.npmjs.com/package/express) - Helper tool for your Nodejs development.

## Installation

```sh
npm install -g exybox
```

## Quick Overview

Create new app:
```sh
npx exybox create:app -t exybox-typescript my-app
cd my-app

```
